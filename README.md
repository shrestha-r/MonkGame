# MonkGame (C++ RPG Prototype)

## Description
MonkGame is a console-based dungeon exploration RPG developed in C++. It demonstrates object-oriented programming and design patterns.

## Features
- Random dungeon generation
- Multiple room types
- Turn-based combat
- Upgrade system
- Win/Lose conditions

## Build Instructions
### commind line (Terminal)
```
g++ -std=c++17 main.cpp src/*.cpp -Iinclude -o MonkGame
./MonkGame
```

### Using CMake:
```bash
mkdir build
cd build
cmake ..
make
./MonkGame
```
## Controls
 - Input numbers to choose actions
 - Follow on-screen prompts

## Requirements
C++17 compiler
CMake (optional)
